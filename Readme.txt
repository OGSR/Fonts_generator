《潜行者》汉化包生成工具

=======================================================================

配置需求

操作系统：Windows XP SP2～Windows 8.1
软件环境：必须安装 .Net Framework 3.5 和 微软雅黑 字体
CPU     ：双核1.8GHz以上，建议四核2.4GHz以上
内存    ：1GB（XP） 1.5GB（Vista及以上） 建议2GB以上
硬盘    ：至少200MB剩余空间 建议300MB以上剩余空间

微软 .Net Framework 3.5 下载地址：
http://download.microsoft.com/download/6/0/f/60fc5854-3cb8-4892-b6db-bd4f42510f28/dotnetfx35.exe

微软雅黑字体在Windows Vista/7及以上版本中,随操作系统提供，无需额外下载安装。
如果你在使用Windows XP，并且系统中没有安装微软雅黑字体，请下载并安装：
http://download.microsoft.com/download/d/6/e/d6e2ff26-5821-4f35-a18b-78c963b1535d/VistaFont_CHS.EXE

如果以上地址无法下载或下载缓慢，请自行搜索其他下载地址。

工具说明

该工具用于《潜行者》系列游戏及其MOD的汉化包生成，可根据config.ini文件的配置，
生成《潜行者》游戏各个版本的字库文件，配置文件的说明已在后面给出。

请将已经汉化完毕的所有XML文件，放置于xmlfiles目录中，然后再运行Generator.bat
注意：XML文件必须使用UTF-8编码！非常重要！（有/无BOM头均可，程序可自动识别）

如果发生错误，请先检查XML文件（尤其是字符编码），以及软硬件环境是否符合需求。
若认为是程序错误，可到潜行者吧报错，并提供错误截图、相关文件及系统环境等信息。

如果没有出错的话，你将会得到gamedata文件夹，该文件夹即是完整的汉化包。
可以将汉化包按照游戏MOD的安装方法安装到游戏中后进行测试，测试无误即可打包发布。

配置文件说明

config.ini是本程序的配置文件，示例如下：
-----------------------------------------------------
; 这是本工具的配置文件
;
; 以';'开头的行是注释，对于程序不起任何作用。
; 请勿在'='的前后加入任何空格或制表符
; 同时也不要在行首及行尾添加任何额外字符

; 语言，可以是chs、cht等，注意只能包含字母和数字
LANG=chs
;LANG=cht

; 游戏版本，只能是以下3者之一，请根据实际情况设置
;GAME=SoC
;GAME=STCS
GAME=CoP
-----------------------------------------------------

请务必正确设置配置文件中的游戏版本，因为不同版本的游戏字库是不能兼容的。

语言选项用于制作多语言共存的汉化包，防止字库文件互相覆盖。
例如制作简繁双语汉化包时，请按如下步骤：
1、将简体汉化XML文件放入xmlfiles中，并设置LANG=chs
2、生成字库
3、将gamedata文件夹重命名为gamedata_chs
4、将繁体汉化XML文件放入xmlfiles中，并设置LANG=cht
5、生成字库
6、将gamedata_chs文件夹中的全部文件覆盖到gamedata文件夹中
这样就产生了一个简繁双语的汉化包。
如需切换语言，只需打开gamedata\config(s)\localization.ltx文件，
同时修改其中的language和font_prefix为chs/_chs或cht/_cht即可。
注意font_prefix需要在语言名称之前加一个下划线字符，否则将导致游戏找不到字库。
